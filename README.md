# My Hyprland Dotfiles
(based on [this](https://github.com/Zproger/bspwm-dotfiles) and [catppuccin palette](https://catppuccin.com/palette))

## Components:
- **OS**: Arch Linux
- **WM**: Hyprland
- **Bar**: Waybar
- **Menu**: Wofi/Rofi
- **Browser**: Firefox
- **Noftication Center**: Swaync
- **Terminal**: Alacritty
- **Shell**: Fish with Starship
